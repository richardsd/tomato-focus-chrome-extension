Shared features
Everything you need to stay in flow.

Core capabilities available across both Chrome and macOS.

⏱
Core timer
25-min work sessions, 5-min short breaks, 15-min long breaks. Circular progress ring, session counter, auto-start, and skip break.
✓
Task management
Create, edit, and delete tasks. Link each session to a task and track estimated vs consumed Pomodoros. Filter by All, In Progress, or Completed.
◈
Jira sync
Import Jira Cloud assignments on demand or on a schedule. Credentials stored securely in local storage — never leaves your device.
◉
Statistics
Daily summary of completed sessions and total focus time. Task-based productivity metrics. Clear stored data whenever you like.
⚙
Flexible settings
Adjust all durations (1–60 min), sessions before a long break, notification sounds and volume, idle detection, and light/dark theme.
▸
Smart notifications
System alerts when sessions end. Idle detection to pause and resume automatically. Quick timers (5/15/25/45 min) for ad-hoc focus blocks.
Installation
Pick your platform.

Both versions are built from the same repository.

Chrome extension coming to Web Store
  1. 1
    Clone or download the repository.
  2. 2
    Open chrome://extensions/ in Chrome (or edge://extensions/ in Edge).
  3. 3
    Enable Developer Mode via the toggle in the top-right corner.
  4. 4
    Click Load unpacked and select the extension folder.
Chrome Web Store — coming soon Once live, one-click installation will be available directly from the Store.
macOS app build from source
  1. 1
    Open in Xcode: open macos-app/TomatoFocusMacApp.xcodeproj and run the TomatoFocusMacAppApp scheme.
  2. 2
    Or build a .app bundle via tools/macos-build-app.sh — output lands in dist/macos/TomatoFocus.app. Drag into /Applications.
  3. 3
    Or build a DMG for distribution: tools/macos-release.sh package-dmg "dist/macos/TomatoFocus.app" dist
Requirements Xcode installed. Swift Package Manager dependencies resolve automatically on first build.
